<p>This document provides important information about the data that we collect
when you use Google Public DNS, how long we store it, and how we use it. You
can also read our <a href="/speed/public-dns/terms">Terms of Service</a>.</p>

<h2 id="our_commitment" data-text="Our Commitment" tabindex="-1">Our Commitment</h2>

<p>We take your privacy very seriously. Google Public DNS adheres to the
<a href="https://policies.google.com/privacy">Google Privacy Policy</a>. Moreover, Google does not use any personal
information collected through the Public DNS service to target ads.</p>

<p>We do not correlate or associate personal information in Google Public DNS
logs with your information from use of any other Google service except for
addressing security and abuse.</p>

<h2 id="information_that_we_collect" data-text="Information that we collect" tabindex="-1">Information that we collect</h2>

<p>The information that we collect about your DNS queries is stored in two types
of logs, each detailed below:</p>

<ul>
<li><a href="#temporary-dns-logs">Temporary DNS Logs</a></li>
<li><a href="#permanent-dns-logs">Permanent DNS Logs</a></li>
</ul>

<h3 id="temporary-dns-logs" data-text="Temporary DNS Logs" tabindex="-1">Temporary DNS Logs</h3>

<p>Temporary logs are the only logs that store both your IP address and your DNS
query.  Specifically, the temporary logs include:</p>

<ul>
<li>the IP address of your device sending the DNS query</li>
<li>the technical information collected for Permanent Logs (see below)</li>
<li>for DNS-over-HTTPS (DoH): the Content-Type and Accept HTTP headers</li>
</ul>

<p>These logs are subject to our deletion processes within 24-48 hours. They are
only used for the following reasons:</p>

<ul>
<li>to identify and mitigate security threats or activity that we deem abusive
or otherwise malicious,</li>
<li>to fix, maintain, and improve Google DNS services, and</li>
<li>to generate aggregate non-personally identifiable DNS usage summaries.</li>
</ul>

<p>To address security and abuse issues, we may retain information from the
temporary logs for longer than 48 hours, in each case solely for the limited
purpose of resolving such issues.</p>

<h3 id="permanent-dns-logs" data-text="Permanent DNS Logs" tabindex="-1">Permanent DNS Logs</h3>

<p>The permanent logs are a sampling of the temporary logs where your IP address
is removed and replaced by a city or region-level location.  Thus, the
permanent logs contain no personal information about you. The following
information is logged in the permanent logs:</p>

<ul>
<li>Requested domain name</li>
<li>Request type (<code translate="no" dir="ltr">A</code> , <code translate="no" dir="ltr">AAAA</code> , <code translate="no" dir="ltr">NS</code>, <code translate="no" dir="ltr">MX</code>, <code translate="no" dir="ltr">TXT</code>, etc.)</li>
<li>Request Size</li>
<li>Transport protocol (<code translate="no" dir="ltr">TCP</code>, <code translate="no" dir="ltr">UDP</code>, <code translate="no" dir="ltr">TLS</code>, or <code translate="no" dir="ltr">HTTPS</code>)</li>
<li>Client&#39;s autonomous system number</li>
<li>User&#39;s geolocation: country, region, and city (no more specific than 1 km²
and 1000 users)</li>
<li>DNS Response code</li>
<li>Google DNS server information</li>
<li>Timestamp</li>
<li>Processing time</li>
<li>Response DNS flags (including <code translate="no" dir="ltr">AD</code>, <code translate="no" dir="ltr">CD</code>, <code translate="no" dir="ltr">DO</code>, <code translate="no" dir="ltr">RD</code>, and <code translate="no" dir="ltr">TC</code>)</li>
<li>Response size</li>
<li>EDNS version</li>
<li>EDNS option</li>
<li>EDNS Client Subnet (ECS) (IP protocol and prefix length -- excluding the
client IP address)</li>
<li>For DNS-over-HTTPS (DoH):
<ul>
<li>Version string corresponding to HTTP path (<code translate="no" dir="ltr">/dns-query</code> or <code translate="no" dir="ltr">/resolve</code>)</li>
<li>Response HTTP encoding, such as <code translate="no" dir="ltr">application/dns-message</code> or <code translate="no" dir="ltr">json</code></li>
</ul></li>
</ul>

<p>You can see <a href="/speed/public-dns/archive/privacy-changes">all past versions</a> of this policy.</p>
