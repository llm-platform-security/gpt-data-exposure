            <section id="available-fields" class="method">
                <div class="method-area">
                    <div class="method-copy">
                        <div class="method-copy-padding">
                            <h3>GET Available Fields</h3>
                            <p>
                                Provides a list of data sets available for a specific vehicle.<br><br>

                                With just basic vehicle identification by VIN or YMM, the only fields that will be valid are:
                                <ul>
                                    <li>Decode</li>
                                    <li>Port</li>
                                    <li>TSB</li>
                                    <li>Maintenance List</li>
                                    <li>Recall</li>
                                    <li>Warranty</li>
                                </ul>

                                To get Vehicle History Report a VIN must be provided.<br /><br />
                                For data on repair or diagnostics, submission of the mileage and dtcs are needed in addition to the vehicle information.<br><br>
                                To check the maintenance and upcoming fields, mileage information is required.
                                For clarity maintenance is strictly maintenance at a certain mileage, while maintenance list is the all the maintenance items for that vehicle.
                                Maintenance requires mileage, while Maintenance List does not.<br><br>
                                Special note regarding repair and diagnostics:  These fields will only work with VIN parameter and will not work with YMM parameters.
                            </p>
                            <p>
                                HTTP REQUEST<br>
                                GET <mark>/fields?vin=&lt;vin&gt;</mark><br>
                                GET <mark>/fields?vin=&lt;vin&gt;&amp;mileage=&lt;mileage&gt;</mark><br>
                                GET <mark>/fields?vin=&lt;vin&gt;&amp;mileage=&lt;mileage&gt;&amp;dtc=&lt;dtc&gt;</mark><br>
                                GET <mark>/fields?vin=&lt;vin&gt;&amp;mileage=&lt;mileage&gt;&amp;unit=&lt;unit&gt;</mark><br>
                                GET <mark>/fields?vin=&lt;vin&gt;&amp;mileage=&lt;mileage&gt;&amp;dtc=&lt;dtc&gt;&amp;unit=&lt;unit&gt;</mark><br><br>

                                GET <mark>/fields?year=&lt;year&gt;&amp;make=&lt;make&gt;&amp;model=&lt;model&gt;</mark><br>
                                GET <mark>/fields?year=&lt;year&gt;&amp;make=&lt;make&gt;&amp;model=&lt;model&gt;&amp;mileage=&lt;mileage&gt;</mark><br>
                                GET <mark>/fields?year=&lt;year&gt;&amp;make=&lt;make&gt;&amp;model=&lt;model&gt;&amp;mileage=&lt;mileage&gt;&amp;unit=&lt;unit&gt;</mark>
                            </p>
                            <p>
                                ENDPOINT DEFINITION<br>
                                GET <mark>http://api.carmd.com/v3.0/fields?vin=&lt;vin&gt;</mark><br>
                                GET <mark>http://api.carmd.com/v3.0/fields?year=&lt;year&gt;&amp;make=&lt;make&gt;&amp;model=&lt;model&gt;</mark>
                            </p>
                            <p>Year, Make, Model, Engine parameter information is provided from <a href="#ymm"> GET Year, Make, Model, Engine </a> endpoints.</p>

            <section id="vin-decode" class="method">
                <div class="method-area">
                    <div class="method-copy">
                        <div class="method-copy-padding">
                            <h3>GET VIN Decode</h3>
                            <p>
                                Decodes a VIN to its year, make, model, and engine.  Covers vehicles from 1981 to current year.
                            </p>
                            <p>
                                This endpoint provides the following data:
                                <ul>
                                    <li>Year</li>
                                    <li>Make</li>
                                    <li>Model</li>
                                    <li>Manufacturer</li>
                                    <li>Engine</li>
                                    <li>Trim</li>
                                    <li>Transmission</li>
                                </ul>
                            </p>
                            <p>
                                HTTP REQUEST<br>
                                GET <mark>/decode?vin=&lt;vin&gt;</mark><br>
                            </p>
